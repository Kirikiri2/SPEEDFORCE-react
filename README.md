# SPEEDFORCE

SPEEDFORCE — это веб-сайт сервиса посуточной аренды автомобилей, разработанный в рамках учебного проекта. Приложение выполнено с современным дизайном и удобным пользовательским интерфейсом, позволяющим пользователям просматривать доступные автомобили, оформлять бронирование и отслеживать статистику аренды.
---

## Стек технологий

### Фронтенд:

- React

- Tailwind CSS

- Vite

### Бэкенд:

- Node.js

- Express

### База данных:

- SQLite3

### Контроль версий:

- GitVerse

## Функциональные возможности

- Каталог автомобилей с разделением по категориям

- Система бронирования автомобилей

- Регистрация и авторизация пользователей

- Просмотр статистики бронирований и аренды

- Адаптивный интерфейс для всех типов устройств
---
